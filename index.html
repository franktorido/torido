##install kali terminal in android termux##

#simply follow these steps:

1. Download termux latest version, click here

2. After downloading simply install and open the app.

##Run these command in termux terminal##

1. termux-change-repo

2. Sellect all repo

3. Select mirrors by Tsinghua U.

4. pkg update

5. pkg upgrade

6. pkg install git

7. copy this link and paste in your termux  curl -LO https://raw.githubusercontent.com/Hax4us/Nethunter-In-Termux/master/kalinethunter

8. chmod +x kalinethunter

9. ./kalinethunter

 ##it will take about 2.5 gb of data##

##after successfully  extracting the files##

run the command startkali

root access command sudo su and use deffault password kali

👉: All these red coloured text are command and keywords you       will use in your termux shell
